# Screenshots da Landing Page

Este diretório contém as screenshots da plataforma que são exibidas na seção "Veja Como é Por Dentro" da landing page.

## Como Adicionar Screenshots

1. **Tire screenshots** das principais telas da plataforma:
   - Dashboard gamificado
   - Página de Missões e Módulos
   - Tela de Simulados
   - Interface dos Professores IA
   - Sistema de Níveis e Badges
   - Página de Competições

2. **Salve as imagens** neste diretório com os seguintes nomes:
   - `dashboard.png` - Dashboard principal
   - `missoes.png` - Página de missões
   - `simulados.png` - Tela de simulados
   - `professores-ia.png` - Interface dos GPTs
   - `niveis.png` - Sistema de níveis e badges
   - `competicoes.png` - Página de competições

3. **Recomendações técnicas:**
   - Formato: PNG ou JPG
   - Resolução: 1920x1080 ou similar (16:9)
   - Tamanho: Máximo 500KB por imagem (otimize antes de adicionar)
   - Qualidade: Alta, mas comprimida para web

4. **Dicas para boas screenshots:**
   - Use dados reais (não deixe campos vazios)
   - Mostre funcionalidades em ação
   - Capture telas com bom conteúdo visível
   - Evite informações sensíveis de usuários reais

## Fallback

Se uma imagem não existir, será exibido um ícone com gradiente como fallback, então não é obrigatório ter todas as imagens de uma vez.
